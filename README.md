# Сравнение Dagster и Airflow

## Airflow если:

Нужен проверенный enterprise-инструмент с огромной экосистемой

Работаем с разнородными системами и редкими коннекторами

Основной фокус — оркестрация задач, а не данные

## Dagster если:

Строим современные data-платформы (dbt, ML, аналитика)

Критически важны lineage данных и observability

Хотим быструю разработку с тестированием из коробки

## Проще говоря:

Airflow — для сложных enterprise-процессов

Dagster — для современных data-команд
